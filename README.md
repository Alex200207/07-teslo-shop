# TesloShop

Pasos rápidos para poner el proyecto en marcha:

1. Clonar el repositorio
  - git clone <url-del-repositorio>

2. Entrar en la carpeta del proyecto
  - cd 07-teslo-shop

3. Instalar dependencias
  - npm install  (o `yarn` / `pnpm` según prefieras)

4. Configurar variables de entorno
  - Copiar `.env.example` a `.env` y completar las claves necesarias (API, base de datos, etc.)

5. Ejecutar en modo desarrollo
  - npm run dev  (o `npm start` según la configuración)

6. Probar la aplicación
  - Abrir http://localhost:3000 (o el puerto indicado) y verificar funcionalidades básicas

7. Construir para producción
  - npm run build
  - npm run start (si aplica)

8. Contribuir
  - Crear una rama por feature: `git checkout -b feature/nombre`
  - Hacer commits claros y abrir un Pull Request
